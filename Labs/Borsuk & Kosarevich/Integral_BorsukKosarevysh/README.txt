config.txt файл для задання параметрів
    alpha           абсолютна похибка
    relalpha        відносна похибка
    xrange          мін. та макс. значення x
    yrange          мін. та макс. значення y
    xdelta          початкова кількість розбиттів по x
    ydelta          початкова кількість розбиттів по y
    m, c, a1, a2    параметри функції

run.py            скрипт основного завдання
additional_run.py скрипт додаткового завдання
