1)
Щоб запустити інтегрування:
g++ -std=c++14 -O3 -ffast-math -pthread multiple_threads.cpp -o multiple_threads && ./multiple_threads [-arguments] [value]
g++ -std=c++14 -O3 -ffast-math -pthread single_thread.cpp -o single_thread && ./single_thread [-arguments] [value]

Аргументи (значення по замовчуванню):
-xmin (-100.0) - нижня межа інтегрування по x
-xmax (100.0) - верхня межа інтегрування по x
-ymin (-100.0) - нижня межа інтегрування по y
-ymax (100.0) - верхня межа інтегрування по y
-relerr (0.001) - відносна похибка
-maxiter (8) - максимальна кількість ітерацій розбиття
-abserr (0.2) - абсолютна похибка
-m (5) - параметр функції
-parts (200) - початкова кількість розбиттів
-maxparts (100000) - максимальна кількість розбиттів

2)
Щоб запустити Python скрипт:
python run.py

Перший аргумент - кількість запусків
Другий аргумент - абсолютна похибка

Можна редагувати файл, щоб запустити з конкретними аргументами
